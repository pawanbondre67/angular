# Housing App

Housing App is a web application built with Angular that helps users find, list, and manage properties.

## Features

- Search and filter properties based on various criteria such as location, price, size, etc.
- View detailed information and images for each property.
- Users can sign up, sign in, and manage their profiles.
- Property owners can list their properties for rent or sale.
- Messaging system for communication between property owners and potential buyers/renters.

## Installation

To run the Housing App locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/housing-app.git

2. cd housing-app
3.   npm install
4.  ng serve
5.  Open your browser and visit http://localhost:4200 to view the app.

##Technologies Used
- Angular: Frontend framework for building the user interface and logic.
- Angular Material: UI component library for creating beautiful and responsive designs.
